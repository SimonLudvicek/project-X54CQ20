Finanční aplikace ve FAB
--------------------------------------------------------------

- aplikaci jednoduše nainstalujete pomocí názorných příkazů

    git clone https://github.com/SimonLudvicek/project-X54CQ20
    pip install -r requirements.txt

- spustíte jí jedoduchým příkazem:

    flask run

- Použité knihovny
    Yfinance
    plotly
    pandas
